# 功能列表

这个功能表是yobot3包含的功能，其他常用功能可以通过安装酷Q插件来实现（插件推荐），有问题可以加QQ群反馈（770947581）。

如果需要给命令添加前缀，请发送“设置”为机器人设置触发前缀。例如：设置前缀为“#”时，只有发送“#状态”才能触发获取boss状态功能。请注意前缀的全角/半角区别。

公会战功能使用手册

(权)标记表示功能需要权限

(自动)标记表示无需触发词，机器人主动推送

(v3.x)标记表示此功能所需的版本号

方括号表示参数可以省略
