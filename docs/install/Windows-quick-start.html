# 安装

# 第一步

点击下载酷Q-httpapi 整合包

解压后双击“CQA.exe”，输入账号密码（注意账号风险，请勿使用重要的QQ号）

右键点击悬浮窗，应用→应用管理→启用HTTP API插件（如果无法启动请参考下方的常见问题）

启动 httpapi 插件后会反复出现如下的提示

# 第二步

点击下载yobot运行包

解压后双击“yobot.exe”启动服务，双方通信成功后出现如下的提示

# 使用

# 常见问题

# Http Api 无法启动

参考：httpapi插件文档

如果你的windows不是最新版，可能无法启动httpapi插件，请安装Visual C++ 可再发行软件包（点击下载）。如果你的系统是 Windows 7 或 Windows Server 2008、或者安装 Visual C++ 可再发行软件包之后仍然加载失败，则还需要安装通用 C 运行库更新（点击进入官网，选择你系统对应的版本下载安装）

# 酷Q的日志显示了发送，实际却没有发送

这种情况一般是消息被腾讯屏蔽，常常发生在异地登陆时

解决方法：

  • 关闭所有插件，机器人挂机一段时间
  • 在远程主机上使用一下QQ网页服务（比如QQ邮箱、QQ安全中心等）
  • 下载QQ安全中心，确认异地登录
