# Windows 运行包（新手推荐）

直接运行 yobot.exe

# Windows 源码运行

用 python 运行源码，可以方便地做一些小改动

# Linux 源码运行

在 Linux 或 MacOS 上用 python 运行源码， 同时在 docker 下使用 wine 运行酷Q。 由于不是原生环境，所以其他插件可能不兼容。

详细说明

# 作为 nonebot 插件运行

在 Windows 或 Linux 或 MacOS 上运行 nonebot， 将 yobot 作为 nonebot 插件运行， 方便扩展与进一步开发。
