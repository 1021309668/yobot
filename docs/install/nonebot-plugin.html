# 作为 nonebot 插件运行

前提：已经安装好nonebot（或者其衍生物）并且可以正常运行

nonebot介绍

进入nonebot插件目录，下载仓库：

git clone https://github.com/yuudi/yobot.git

（或者使用国内源https://gitee.com/yobot/yobot.git）

（或者使用git的子模块来管理git submodule add https://......）

重新加载nonebot插件，开始使用
